## Статус проекта (портфолио-срез)

Этот репозиторий — **не финальный релиз**. Это **портфолио-срез** проекта на конкретный момент разработки (код + контент, созданные мной).

**Здесь можно встретить:**

* незавершённые фичи и TODO,
* временные решения / обходные пути,
* известные и неизвестные баги.

Цель репозитория — показать мой код, структуру проекта и ключевые системы, а не отполированную продакшен-сборку.

## Реализованный функционал

* **Полная поддержка мультиплеера**

  * Основано на OnlineSubsystemSteam
  * Работа с лобби
  * Поток матча: Лобби → Игровая сессия

* **Gameplay Ability System (GAS)**

  * Способности, атрибуты, эффекты и состояние на базе gameplay tags

* **Инвентарь и экипировка**

  * Управление инвентарём
  * Система экипировки

* **ИИ-противники**

  * Враг ближнего боя
  * Враг дальнего боя
  * Босс с фазами, зависящими от процента здоровья

* **Боевая система**

  * 3 типа оружия / 3 стиля боя
  * Переключение между стилями “на лету”
